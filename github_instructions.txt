## GitHub Repository Setup Instructions

1. Create a new repository on GitHub:
   - Go to: https://github.com/new
   - Repository name: LOCKDOWN-Escape-Room
   - Description: A web-based escape room 3D interactive application built with Three.js
   - Choose Public or Private visibility
   - Do NOT initialize with README, .gitignore, or license
   - Click "Create repository"

2. Connect your local repository to GitHub:
   - Run the following commands in your terminal:

```
git remote add origin https://github.com/x71xxy/LOCKDOWN-Escape-Room.git
git branch -M main
git push -u origin main
```

Note: Replace "x71xxy" with your actual GitHub username in the URL.

3. When prompted, enter your GitHub username and password or personal access token.

4. After successful push, your code will be available at:
   https://github.com/x71xxy/LOCKDOWN-Escape-Room

5. To view your GitHub Pages site (if enabled):
   - Go to your repository settings
   - Scroll down to GitHub Pages section
   - Select source branch (main) and save
   - Your site will be published at:
   https://x71xxy.github.io/LOCKDOWN-Escape-Room/
